# NpyViewer
You just need to download the.exe file and then set the default opening path of.npy to this .exe program. The python code in the repository is the source code of the program.
The version of vscode I'm using doesn't support the npy visualization plugin in vscode, and I don't want to use python code for visualization. Otherwise, I would have to modify the file path every time I view an npy file. So I made this simple program with AI.

你只需要下载.exe文件，然后将.npy的默认打开路径设置为该exe程序。仓库中的python代码是exe的源代码。
